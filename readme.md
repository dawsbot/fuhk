> returns all &#34;bad words&#34; found in a javascript variable

<br>



## Install

```
npm install --save fuhk
```


<br>

## Usage

```js
const fuhk = require('fuhk');

fuhk('hackathons');
//=> 'hackathons & rainbows'
```

<br>

## API

### fuhk(target)

##### target

Type: `string` || `Array`

returns all &#34;bad words&#34; found in a javascript variable
